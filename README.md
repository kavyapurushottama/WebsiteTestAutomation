## Description

Automated UI testing project for an e-commerce web application using Selenium
with Python and PyTest. Designed manual test cases and automated critical user
flows such as login, product visibility, and add-to-cart functionality, with test
execution reports generated using PyTest-HTML.

## Tools Used

- Selenium WebDriver
- Python
- PyTest
- ChromeDriver
- PyTest HTML Reports

## Test Scenarios Automated

- Login functionality
- Product visibility validation
- Add to cart functionality
- Basic regression testing

## Framework Design

- Basic Page Object Model (POM) structure
- Modular test organization
- Reusable test methods
